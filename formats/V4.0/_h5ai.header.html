VRDC Data @ Cornell

Experimental QWIPU Schema files

See QWIPU_Data_Schema.pdf (downloaded from http://lehd.ces.census.gov/doc/QWIPU_Data_Schema.pdf on 2014-10-30) for more details. A similar document, created by Lars Vilhuber (Cornell University), with download links to each of the machine-readable CSV files in this directory, is provided in lehd_public_use_schema.html. To download this entire directory, please go up one level, click the checkbox next to the directory, and choose "download" from the top right menu.
