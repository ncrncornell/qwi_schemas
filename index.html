LEHD Experimental Schemas

Experimental LEHD Schema files, that may or may not later be incorporated into public-use LEHD file and content specifications. Currently covers QWI, J2J, and shapefiles.

These are NOT official publication standards of any agency. However, they may turn out to be useful.

For official LEHD-published schema, see http://lehd.ces.census.gov/data/schema/

You can browse the following schemas:

  • v3.5: Simply a store (incomplete) of the previous version.
  • V4.0 released versions
    • V4.0: A translation of the current version in use at http://lehd.ces.census.gov/data for QWI.
    • V4.0.1: First schema in this proposed format, including updates to geography files for R2015Q2 data releases by the Census Bureau, LEHD Program.
    • V4.0.2: An update to 4.0.1 reflecting change of industry coding to NAICS 2012.
    • V4.0.3: An update to 4.0.2 reflecting geography changes for R2015Q4
    • V4.0.4: An update to 4.0.3 reflecting geography changes for R2016Q1
  • V4.1 released versions
    • V4.1.0: The released V4.1.0 version.
    • V4.1.1: The released V4.1.1 version.
    • V4.1.2: The released V4.1.2 version.
    • V4.1.3: The released V4.1.3 version.
  • V4.2 released versions
    • V4.2.0: The released V4.2 version.
  • V4.3 versions
